Peformance Testing:
=================
Simulate real business use cases of your web app
Test the app under the predefined concurrency
Validating the app/servers readiness ahead of peak events
Test the app before/after version release to ensure the stability

Jmeter - tool
=============
Its a java based application
A synchronous, thread based, script execution engine
Can be utilized for load and performance testing
Fully OpenSource(Apache 2.0 license) project
Not a browser although simulates some of a browser behaviour
